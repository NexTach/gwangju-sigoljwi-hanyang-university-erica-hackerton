# Road DNA

휠체어·유모차 이동 중 스마트폰 센서 신호를 기기에서 분석하고, 검증된 충격
후보만 도로 구간 단위로 집계하는 접근성 크라우드센싱 플랫폼입니다.

> 이동의 흔적이 도시의 장벽을 발견하다.
