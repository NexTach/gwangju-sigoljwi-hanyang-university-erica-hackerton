# Login brand assets

These files are the official login-brand resources used by the demo login
screen. Keep their color, shape, and proportions unchanged.

- `google-g.png`: Google Identity, Sign in with Google branding resources
  - Source: https://developers.google.com/static/identity/images/g-logo.png
  - Guide: https://developers.google.com/identity/branding-guidelines
- `kakao-login-large-wide.png`: Kakao Developers standard Korean login button
  - Source: https://developers.kakao.com/tool/resource/static/img/button/login/full/ko/kakao_login_large_wide.png
  - Guide: https://developers.kakao.com/docs/ko/kakaologin/design-guide
  - The mobile login screen exposes the symbol area from this official bitmap
    at its original proportions; it does not substitute a chat or KakaoTalk icon.
