# Road DNA 개인정보·데이터 보관 정책

## 최소 수집

서버는 이름, 이메일, 전화번호, 계정, 주소록, 사진을 요구하지 않는다. 앱이
처음 만든 익명 UUID, 이동 유형, 세션 시각, 충격 후보의 위치·정확도·요약 특징만
수집한다. 이 전송은 앱의 `익명 이동 데이터 공유`가 켜진 경우에만 수행한다.

수집하지 않는 데이터:

- 전체 가속도·자이로 원본 스트림
- 사용자의 연속 이동 경로
- 집·직장 등 출발지와 목적지 의미 정보
- 장애물 종류를 확정하는 사진·음성

## 기기 내 처리

가속도와 자이로를 2초 창으로 필터링하고 특징을 계산한다. 평상시 창은 폐기하며
후보가 생긴 경우에만 해당 시점의 GPS와 요약 특징을 API로 보낸다. 공유를 끄거나
API가 응답하지 않으면 결과는 기기 안에서만 처리한다. 지도에는 개별 사용자 경로가
아니라 도로 구간 집계만 노출한다.

## 보관과 삭제

- 상세 후보 이벤트·세션·도로 통과 연결 정보: 기본 90일
- 이동유형별 도로 점수·신뢰도·집계 건수: 통계 레코드로 유지
- `EVENT_RETENTION_DAYS`는 30~3650일 범위에서만 설정 가능
- GitHub Actions가 매일 03:17 KST에 만료 상세 레코드를 트랜잭션으로 삭제
- 상세 레코드가 없는 익명 UUID도 함께 삭제

운영자는 `node dist/scripts/cleanup.js`를 실행해 같은 정리 작업을 수동으로
검증할 수 있다. 삭제 후에도 집계 점수는 남으므로 개인 수준의 과거 경로를
복원하지 않고 도시 단위 추세를 유지한다.

## 표현 원칙

- 센서 후보를 확정된 턱·포트홀로 표시하지 않는다.
- Road DNA Score는 실험적 내부 지표임을 항상 함께 노출한다.
- 위치 데이터가 포함되므로 “완전히 개인정보 문제가 없다”고 주장하지 않는다.
- 실제 운영 전 개인정보 처리방침, 삭제 요청 창구, 법적 보관 근거를 별도로
  검토한다.
